The second way to add date validation is using the matDatepickerFilter property of the datepicker input. This property accepts a function of D => boolean (where D is the date type used by the datepicker, see Choosing a date implementation). A result of true indicates that the date is valid and a result of false indicates that it is not. Again this will also disable the dates on the calendar that are invalid.
